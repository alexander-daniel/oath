Oath

The Trustkeeper for Echoes of the Lake

1. Register

Endpoint: POST https://auth.echoesofthelake.com/register

Description: Register a new user by providing a username and password.

Request

{
  "username": "your-username",
  "password": "your-password"
}

Response

Status: 201 Created
{
  "message": "User created successfully"
  "token": "your-jwt-token"
}

2. Login

Endpoint: POST https://auth.echoesofthelake.com/login

Description: Authenticate a user and receive a JWT token for secure access.

Request

{
  "username": "your-username",
  "password": "your-password"
}

Response

Status: 200 OK
Body: {
  "token": "your-jwt-token"
}

3. Validate Token

Endpoint: GET https://auth.echoesofthelake.com/validate

Description: Validate the authenticity of a JWT token.

Request

Headers:
Authorization: Bearer your-jwt-token

Response

Status: 200 OK
Body: {
  "valid": true
}
